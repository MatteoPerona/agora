# Agora Frontend

This is the canonical Agora frontend: the debate design system app that now ships as the only web UI for the project.

## Running the code

```bash
npm install
npm run dev
```

The Vite dev server runs on `http://localhost:5173` and proxies `/api` and `/health` to the backend on `http://localhost:8000`.

## Build

```bash
npm run build
```
